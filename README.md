Pyloton: CircuitPython Cycling Computer

Open Source cycling computer that displays heart rate, speed, cadence, and song playback info!

https://learn.adafruit.com/pyloton

To stream sensor data from a Garmin sensor to Matlab or Python, it has in Matlab an instruction which is probably easy to make real for PhD in computer engineering. The Adafruit instruction CircuitPython BLE Heart Rate Zone Trainer Display is a 33 pages when printed it out on a PC printer. With Bitalino, you can pair an ECG devise via bluetooth to a PC running Matlab or Python directly. A Garmin sensor is difficult to connect to a PC.
